# Lead Rescue Admin OS

Static prototype of the "Stop Losing Leads" AI admin automation package for existing small businesses.

## What It Does

- Captures a lead from WhatsApp, Facebook, LinkedIn, Google Business, phone, or referral.
- Scores urgency and job value.
- Generates a WhatsApp reply.
- Builds a quote evidence checklist.
- Creates a follow-up sequence.
- Produces a daily owner summary.
- Frames invoice chasing and review requests as the next package modules.
- Stores captured leads locally in the browser until it is connected to a shared CRM or database.
- Exports a CSV pipeline.
- Generates service-specific post copy.

## Recommended Recovery Metrics

Run it for 7 days against existing and new enquiries, then track:

- Enquiries captured
- Old leads reactivated
- Replies sent within 30 minutes
- Qualified leads
- Site visits booked
- Quoted value
- Jobs won
- Follow-ups recovered
